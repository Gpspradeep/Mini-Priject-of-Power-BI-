# 🚲 Smart Bike Mobility Analytics & Operational Performance Tracker

---

## 📌 Project Overview

This Power BI project focuses on analyzing smart bike-sharing operations, station performance, bike availability, and urban mobility infrastructure across multiple cities. The project transforms raw mobility and operational data into meaningful business insights using Power BI dashboards, data modeling, DAX calculations, and interactive visualizations.

The dashboard helps monitor bike availability trends, station utilization, operational efficiency, city-wise bike distribution, and smart mobility performance through dynamic reporting and KPI analysis.

---

# 🎯 Project Objectives

* Analyze bike availability trends across stations
* Monitor open and closed station performance
* Track quarterly bike availability growth
* Evaluate bike stand capacity utilization
* Compare city-wise bike availability performance
* Measure operational efficiency using KPI analysis
* Identify high-performing and low-performing stations
* Generate actionable business insights using data visualization

---

# 📊 Dashboard Highlights

* 🚲 Bike Availability %: **32.76%**
* 🚴 Total Available Bikes: **13K**
* 🅿️ Total Available Stands: **22K**
* 🏢 Total Stations: **2K**
* ✅ Open Stations: **70.43%**
* ❌ Closed Stations: **29.57%**
* 📈 Highest Quarterly Availability: **11.9K in Q4**

---

# 📌 Key Insights

* Open stations contributed more than **70%** of operational activity across the mobility network.
* Brussels recorded the highest bike availability with nearly **3.5K available bikes**.
* Quarterly bike availability peaked at **11.9K bikes during Q4**.
* Top-performing stations maintained stand capacities between **39–59 units**.
* Bike availability efficiency remained at **32.76%**, highlighting opportunities for operational optimization.
* City-wise analysis showed stronger mobility performance in Brussels, Leuven, and Toulouse.
